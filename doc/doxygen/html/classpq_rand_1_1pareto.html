Detailed Description

Sample the Pareto distribution (with minimum value \( x_\text{min}^{} > 0 \) and index \( \alpha > 0 \)).

\( \text{PDF}(x) = (\alpha\,x_\text{min}^\alpha)/(x^{\alpha + 1}) \)

Note
This distribution has an invertible CDF, but it only becomes ill-conditioned in one tail \((u\to 0)\), so we don't need a quantile flip-flop. However, we still need to use uneven \( U(0, 1] \).
