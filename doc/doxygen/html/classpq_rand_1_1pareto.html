Detailed Description

Samples from the Pareto distribution (for minimum value \( x_m^{} \) and index \( \alpha \)): \( \text{PDF} = (\alpha\,x_m^\alpha)/(x^{\alpha + 1}) \).

This distribution has an invertible CDF: \( \text{CDF}^{-1} = x_m^{} u^{-1/\alpha} \). This \( \mathrm{CDF}^{-1} \) becomes ill-conditioned in only one tail \((u\to 0)\), so we don't need a quantile flip-flop. However, we still need to use quasiuniform \( U(0, 1] \).
