Detailed Description

Sample the uniform distribution over the closed interval [min, max].

\( \text{PDF}(x) = \frac{1}{\text{max}-\text{min}} \)

Warning
Both endpoints are returned (unless a is zero, in which case exactly zero will probably not be returned).
