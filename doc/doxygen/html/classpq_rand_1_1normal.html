Detailed Description

Samples from the normal distribution (mean \( \mu \) and standard deviation \( \sigma \)): \( \text{PDF} = \frac{1}{\sigma\sqrt{2\pi}} \exp\left(-\frac{1}{2\,\sigma^2}(x - \mu)^2\right) \).

normal extends standard_normal by redefining GenTwo() to return \( x = \mu + \sigma\, x_{\mathrm{standard}}^{} \).
