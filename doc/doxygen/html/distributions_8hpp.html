Detailed Description

Defines the sampling objects which draw from various distributions.

The sampling objects use a polymorphic interface which adds functions as more analytic information is known about their distribution. Most distributions utilize a quantile flip-flop (pqRand::distributionQ2, as described in the research paper) but some do not need one (e.g. pqRand::pareto). Others use rejection sampling (e.g. pqRand::standard_normal, pqRand::gammaDist).
