Detailed Description

Samples from the weibull distribution (with shape parameter \( k \) and scale parameter \( \lambda \)): \( \text{PDF} = \frac{k}{\lambda} \left(\frac{x}{\lambda}\right)^{k-1} \exp\left(-(x/\lambda)^k\right)\).

This distribution has an invertible CDF: \( \text{CDF}^{-1} = \lambda(-\log(1-u))^{1/k} \). This \( \mathrm{CDF}^{-1} \) becomes ill-conditioned in both tails, and therefore uses a quantile flip-flop.
