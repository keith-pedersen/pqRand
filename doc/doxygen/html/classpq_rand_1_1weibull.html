Detailed Description

Sample the Weibull distribution (with scale \( \lambda > 0 \) and shape \( k > 0 \)).

\( \text{PDF}(x) = \frac{k}{\lambda} \left(\frac{x}{\lambda}\right)^{k-1} \exp\left(-(x/\lambda)^k\right)\)
