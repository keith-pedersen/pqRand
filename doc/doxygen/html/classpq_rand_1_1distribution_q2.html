Detailed Description

An abstract class which extends the distributionCDF interface for those distributions with an invertible CDF (i.e. quantile functions).

This permits a generic operator() using a quantile flip-flop (the precise inversion method, as discuseed in the theory paper), which keeps the quantile function well-conditioned in both tails.
