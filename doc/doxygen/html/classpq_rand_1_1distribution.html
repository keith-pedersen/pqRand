Detailed Description

template<typename T>
class pqRand::distribution< T >

An abtract (pure virtual) class for sampling random variates.

This class defines a standard interface: the operator() is used to sample one variate, of which the min/max variate must be defined. This permits a DRY (don't repeat yourself) GetSample().
