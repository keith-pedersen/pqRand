Detailed Description

template<typename int_t>
class pqRand::uniform_integer< int_t >

Sample integers uniformly from the half-open [min, max).

Note
max() is not actually returned; the largest sampled variate is (max - 1). This is a slight perversion of the distribution interface, but is most consistent with the standard definition of uniform integer sampling.
