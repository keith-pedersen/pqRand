Detailed Description

Establishes pqRand and defines the PRNG engine needed by distributions.

The important methods implemented here are the "quasiuniform sample" of the semi-inclusive unit interval (pqRand::engine::U_Q) and the random coin-flip (pqRand::engine::RandBool).
