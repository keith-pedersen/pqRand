Detailed Description

Samples from the log-normal distribution: \( \text{PDF} = \frac{1}{x\,\sigma\sqrt{2\pi}} \exp\left(-\frac{1}{2\,\sigma^2}(\log x - \mu)^2\right) \).

log_normal extends normal by redefining GenTwo() to return \( x = \exp (x_{\mathrm{normal}}^{}) \).
